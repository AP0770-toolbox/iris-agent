# Iris Agent

Tiny GitHub Pages site for TikTok Developer sandbox review.

## Pages

- https://AP0770-toolbox.github.io/iris-agent/
- https://AP0770-toolbox.github.io/iris-agent/terms.html
- https://AP0770-toolbox.github.io/iris-agent/privacy.html
- https://AP0770-toolbox.github.io/iris-agent/callback.html

Iris Agent is described as a private TikTok creator analytics assistant, not a posting automation tool. `callback.html` is a static OAuth callback helper that displays TikTok's authorization code locally in the browser and does not contain client secrets.
